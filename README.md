# PixelCraft Agency - Design Homepage

A responsive and modern Design Agency homepage built using Next.js and Tailwind CSS.
The project focuses on clean UI/UX, smooth navigation, and a consistent design system.

---

## ⚙️ Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/Madhu123sudhan/pixelcraft-agency.git
```

2. Navigate to the project directory

```bash
cd pixelcraft-agency
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

5. Open in browser

```
http://localhost:3000
```

---

## 🛠️ Tech Stack Used

- Next.js (App Router)
- React
- Tailwind CSS (v4)
- Framer Motion (for animations)
- next-themes (for dark/light mode)

---

## 📌 Assumptions & Additional Features

### Assumptions

- This project focuses on frontend UI/UX implementation
- Contact form simulates submission (no backend/email integration)
- Images are stored locally in the public folder

### Additional Features

- Dark and Light mode toggle with smooth transitions
- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Animated Hero section using Framer Motion
- Interactive Services and Portfolio sections
- Clean and modern UI with consistent styling
